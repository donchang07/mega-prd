## 1. 목표

PRD v1.2에서 신규·재설계된 화면을 실제 동작하는 콕핏 프로토타입으로 구현한다.
PRD가 명시한 대로 인맥(CRM)·운동 탭은 "원본 데이터는 확보됐으나 DB 미반영" 상태이므로,
개인정보 실데이터 파싱은 범위에서 제외하고 **PRD 화면정의를 충실히 반영한 대표 샘플 데이터**로 화면·상호작용을 완성한다.
